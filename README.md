# Capitask

`Capitask` — open-source сервис управления задачами для команд (аналог Jira) с лёгким UI и фокусом на скорость работы с backlog и sprint.

## Статус проекта

- Лицензия: `MIT` (см. `LICENSE`)
- Модель: open-source
- Текущий стек фронтенда: `React + Vite`

## Что уже реализовано

- Доска задач (статусы `To Do / In Progress / Done`) с drag-and-drop.
- Список задач с фильтрами и поиском.
- Создание и редактирование задач в модальном окне.
- Sprint-экран с редактированием статусов и CSV-экспортом.
- Сохранение состояния в `localStorage`.
- Пасхалка: при прокрутке списка задач до конца появляется капибара.

## Быстрый запуск

Требования:

- `Node.js 18+`
- `npm 9+`

Команды:

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`.

## Скрипты

```bash
npm run dev      # запуск dev-сервера
npm run build    # production-сборка
npm run preview  # локальный просмотр production-сборки
```

## Безопасность

Проект ведётся с обязательным контролем безопасности кода:

- валидация и санитизация входных данных;
- минимально необходимые права доступа;
- безопасная обработка ошибок без утечки чувствительных данных;
- проверка зависимостей и регулярные обновления;
- обязательный review изменений, влияющих на auth, permissions, хранение данных.

Если вы нашли уязвимость, создайте issue с пометкой `security` и **не публикуйте exploit-подробности в открытом виде**.
